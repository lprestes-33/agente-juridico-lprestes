# Agente Jurídico L.Préstès

Sistema pessoal para estudo jurídico, OAB, concursos e produção de peças.

## Pastas

- entrada: textos, PDFs convertidos e materiais para análise.
- saida: resumos, peças e relatórios gerados.
- regras: regras de atuação do agente.
- modelos: modelos de peças.
- pecas: peças processuais.
- estudos: planos de estudo e revisões.
- jurisprudencia: julgados e entendimentos.
- scripts: automações.
